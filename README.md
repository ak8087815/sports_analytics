# Sports Analytics: Outcome Prediction

This repository contains a Machine Learning project focused on predicting future player performance in sports based on historical data and current conditions.

## Project Overview

The project utilizes a dataset of player records to train and evaluate predictive models. It explores various factors such as player age, fitness level, match conditions, and opponent strength to estimate future performance outcomes.

## Dataset

The project uses `Sports_Analytics_100_Records.csv`, which includes the following features:

- **Player Identity**: Player ID and Name (removed during preprocessing)
- **Team Info**: Team Name
- **Demographics**: Gender, Age
- **Performance Metrics**: Matches Played, Runs Scored, Previous Average
- **Contextual Factors**: Fitness Level, Opponent Strength, Match Condition
- **Target Variable**: Future Performance

## Technologies Used

- **Python**: Core programming language.
- **Pandas & NumPy**: For data manipulation and analysis.
- **Scikit-learn**: For machine learning models (Linear Regression, Random Forest).
- **Jupyter Notebook**: For interactive development and visualization.

## Files in this Repository

- `MLproject.ipynb`: The main Jupyter notebook containing data preprocessing, model training, and evaluation.
- `Sports_Analytics_100_Records.csv`: The dataset used for the project.
